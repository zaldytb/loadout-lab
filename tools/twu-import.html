TWU Data Importer — Help

What to paste

Paste any content from Tennis Warehouse University pages. Claude handles all formats:

  • HTML comparison tables (<table class="simtable">)
  • Visible text copied from TWU comparison or review pages
  • Raw HTML from View Source or browser DevTools
  • Mixed content from multiple TWU pages — paste more and click Extract again, rows append
  • Full page source (Ctrl+U / Cmd+U) from a TWU comparison page — contains ALL racquets in the dropdown, not just the two shown. One paste can yield dozens of racquets.

Fields Claude can usually extract

FieldSource
nameRacquet/string name in table header or review title
headSizeTWU comparison table; auto-converts cm² → sq in
strungWeightTWU weight row; auto-converts oz → grams
balanceTWU balance row; auto-converts inches → cm
swingweightDirect from TWU table
stiffness"Flex" row in TWU frame table (RA/RDC rating)
yearFrom review title or product name when present

Fields often missing — fill in manually

beamWidthRarely in comparison tables; check manufacturer spec sheet
patternSometimes in TWU spec, sometimes not listed
tensionRangeManufacturer spec, not always on TWU pages

Cell highlighting

Red borderRequired field — row excluded from CSV until filled in
Orange borderValue outside typical range — warning only, still exported
Yellow backgroundClaude returned null for this optional field — consider filling in

Workflow

  1. Select mode (Frames or Strings) and enter your Anthropic API key
  2. Paste TWU content and click Extract Data
  3. Fill in any red-border cells (required fields Claude couldn't find)
  4. Click Download CSV — only complete rows are included
  5. Run: node pipeline/scripts/ingest.js --type frame --csv path/to/twu-frames-YYYY-MM-DD.csv
  6. Run: npm run pipeline to validate and regenerate data.js

API key security

Your key is stored only in your browser's localStorage (when “Remember key” is checked) and sent exclusively to api.anthropic.com. It is never included in downloaded CSV files or logged to the browser console.
