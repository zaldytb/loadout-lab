Frame Batch Editor — Help

Keyboard shortcuts

Tab / Shift+TabMove to next / previous cell
EnterMove to same column in next row (adds row if on last)
EscapeRevert cell to value before editing started
Click rowSelect row (enables Duplicate / Delete)

Validation indicators

Red borderError — row is excluded from CSV export until fixed
Orange borderWarning — value is outside typical range but row is still exported
Column header *Required field

Workflow

  1. Fill in rows — or load an existing CSV / paste from a spreadsheet
  2. Fix any red-border cells (hover for tooltip explaining the error)
  3. Click Download CSV — only valid rows are included
  4. Run: node pipeline/scripts/ingest.js --type frame --csv path/to/frames-batch-YYYY-MM-DD.csv
  5. Run: npm run pipeline to validate and regenerate data.js
